# Metallic.V1 Landing Page

Standalone landing-page-only package.

## Included
- `index.html` — Metallic.V1 landing page
- `assets/` — only the assets referenced by the landing page

## Not Included
- Business site pages
- Prelaunch/countdown site
- Developer/build/API pages
- Screenshots, PDFs, uploads, duplicate exports

## Deploy
Upload this folder to GitHub, Vercel, Netlify, or any static host. The homepage is `index.html`.
